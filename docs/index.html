Open Teleprompter

A free teleprompter for anyone on camera or on stage.

Free desktop teleprompter for anyone who speaks on camera or on stage. Works offline. Optional AI formatting uses your API key only — never required, never shipped in the code.

What you get

  • Script + prompter modes — paste, format, scroll
  • Local format — no account, no key
  • Optional AI format — Anthropic key you paste in Settings
  • Speed, mirror, fullscreen — second-screen ready

Downloads & source

  • Windows portable + installer — GitHub Releases
  • Full source — github.com/nerdfreakuser/open-teleprompter

Builds may be unsigned. Windows or Chrome can show a warning until a code-signing certificate is used. Building from source is always an option.

Privacy

  • No API keys in the repository
  • Scripts stay on your machine
  • Your optional AI key is stored locally and only sent to the AI provider when you use AI Format
